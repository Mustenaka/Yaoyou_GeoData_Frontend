# 版本更新日志

## 0.1.0 - 2026-07-09

### 账户与企业

- 统一账户登录、角色识别与后台准入控制。
- 企业管理支持企业资料、状态、有效期与授权策略维护。
- 用户管理支持账号状态、角色、企业归属与有效期维护。

### 设备与授权

- 设备指纹台账记录移动端与 Win 端设备身份、归属、状态与风险等级。
- 设备授权支持长期有效与定期有效，到期后保持不可用，需管理员显式调整。
- 授权审批支持新增设备与换机申请处理，并展示企业、用户与设备关联信息。

### 风险管控

- 风险设备识别尝试破解、频繁登陆、频繁切换 IP 与频繁切换账号。
- 风险处理支持标记处理、标记封号与标记封设备，并保留备注与审计记录。
- 安全风险列表与设备风险列表按角色和企业隔离展示。

### 项目与数据

- 项目与数据云归档支持移动端项目包纯上行、后台解析展示与文件下载。
- 项目档案支持按企业、设备、项目逐级查看配置、填充数据与同步文件。
- 移动端功能设置支持智能填充可见性配置。

### 内容与系统

- 内容管理支持官网首页文案、官网图片与技术支持页文案维护。
- 系统设置支持基础配置、邮件配置、系统日志与服务器时间查看。
- 系统信息展示后端服务版本、后台前端版本、构建时间与版本接口状态。
